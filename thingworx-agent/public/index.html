Local-first • generates ThingWorx entities + Extension ZIPs

Build ThingWorx faster with one focused tool.

This UI helps you generate common ThingWorx entities (Projects, Things, ThingTemplates, DataShapes, Mashups) and package them into a valid Extension ZIP. It also includes a documentation + project Q&A flow so you can search context and chat with it.

Features

Designed like a simple “one clear action” flow

Entity generator Generate Project, DataShape, ThingTemplate, Thing, Mashup XML in a consistent export format.
Extension ZIP output Bundles Entities/ and metadata into a valid Extension ZIP you can import into ThingWorx.
Search + Q&A Ask questions, retrieve relevant context, and get concise answers grounded in your docs.

Why it helps

Less guessing, more shipping

Consistency beats complexity Instead of maintaining ad-hoc XML snippets, you start from a predictable structure and iterate.
Small wins add up Generate one entity, download one ZIP, import, verify. Repeat until your model is complete.

How to use

A quick, repeatable loop

  1. Describe what you want to build (use case + entities).
  2. Generate entities and review the produced XML.
  3. Download the Extension ZIP and import into ThingWorx.
  4. Use search + chat to verify details and find references in your docs.

Run locally

Typical local dev flow

cd thingworx-agent
npm install
npm start
